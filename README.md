# How to Run
1. In terminal window, run
    `ng serve`
2. In a separate terminal, run
    `json-server --watch db.json`
2. Go to [http://localhost:4200/](http://localhost:4200/)


# Features
- Home page
    - Displays all habits
- Habit page
    - Details
    - Calendar


# Uses
- Typescript, CSS/HTML
- JSon server database
